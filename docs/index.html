What is HeadWater?

HeadWater is a utility-first CSS system inspired by modern workflows, but designed to stay simple, portable, and build-time friendly. You can ship it as-is, or generate a perfectly trimmed CSS bundle.

HWC ‚Äî HeadWater Components

HWC is a lightweight component language built on top of HeadWater utilities. It lets you define reusable components using class compositions, then compile them into minimal CSS. 

// button.hwc
$btnBase = p-2 rounded inline-flex items-center

.btn = $btnBase bg-primary text-white
.btn:hover = bg-primary-700
.input = p-2 border rounded focus:ring-2 focus:ring-primary

CSS Variants

Lite HW

Ultra-minimal build for landing pages and small projects.

  • 4 colors (primary, secondary, light, dark)
  • Very limited typography
  • Interactive: active only
  • Smallest possible size

Basic HW

Safe default for most projects.

  • 7-color palette
  • Core typography
  • Interactive: hover, focus, active
  • No heavy features

Extra HW

Extended utilities without going fully heavy.

  • Full color palette
  • Typography + selected advanced features
  • Important interactive variants
  • Balanced size vs power

Max HW

Everything enabled. No compromises.

  • All colors
  • Full typography & advanced typography
  • All interactive variants
  • Animations, filters, gradients

Install via npm

Use HeadWater as a generator in your build process. 

npm install @uniantaradev/headwater
hw init
hw build
hw scan .
// generate css
import { HeadWaterGen } from "@uniantaradev/headwater";

const css = HeadWaterGen.generateCSS({
  interactive: { only: ["hover", "focus"] },
  typography: true,
  typographyExtended: false
});
